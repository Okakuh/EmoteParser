- Кидаете архив, или папку ресурспака в папку со скриптом.
- Запускаете скрипт
- Он парсит эмоуты
- Добавляет их в модифицированую версию пака
- Если изначальный пак был архивом то автоматически архивирует его
- Вы также можете включить автоархивацию даже если вы закидываете ресурспак не как архив изначально
- Скрипт обрабатывает только первую попавшуюся папку/архив которая не являеться его конфигом так что не храните лишние папки в папке со скриптом кроме его конфигов
- Настраивайте групы как вам угодно через конфиг
```bash
{
    "get_emotes_data_from": "default.json", ------------------------- # Имя файла из которого парсить символы эмоутов и имена
    "emotes_dir": "assets/myemotes/textures/font/emotes", ----------- # Где храняться картинки эмоутов в паке
    "how_to_define_if_emote": "myemotes",---------------------------- # Последовательность символов которую должен найти скрипт чтобы определить эмоут это или нет
    "tab_width": 10,------------------------------------------------- # Количество табов в симбол чате учитывая таб ппл пака
    "wide_width_to_height": 1.6,------------------------------------- # Соотношение ширины до высоты эмоута для определения широких эмоутов
    "group_prefixes": ["pwgood", "peepo", "pepe", "forsen", "feels"], # Префиксы-имена груп эмоутов
    "non_grouped_emotes_group_name": "other", ----------------------- # Имя групы эмоутов без префикса
    "wide_emotes_group_name": "wide",-------------------------------- # Имя для групы широких эмоутов
    "symbol_chat_exemple_dir": "symbol-chat.exemple",---------------- # Папка-пример для симбол чата
    "where_to_save_symbol_chat_exemple": "assets",------------------- # Куда копировать папку-пример симбол чата
    "result_name": "peepo",------------------------------------------ # Итоговое название файла с сохраненными эмоутами
    "archive_if_was_not?": False ------------------------------------ # Архивировать ли файл если изначально была найдена просто папка?
}
```
